<h1>GDD - Gerenciamento De Dados</h1>

<p>Sistema de gerenciamento de dados para grupos de membros<p>
<h2> serviços: </h2>
<ol>
    <li> Membros: cadastro edição e exclusão </li>
    <li> Relatórios: cadastro, edição e exclusão </li>
    <li> Dashboard: informações do grupo </li>
    <li> Backups </li>
</ol>
<h2> Tecnologias: </h2>
<ol>
    <li> Laravel </li>
    <li> Bootstrap </li>
    <li> HTML5 e CSS </li>
    <li> Javascript e Jquery </li>
</ol>
